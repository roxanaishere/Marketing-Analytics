# Marketing-Analytics

This repository is all my coursework related with marketing analytics, including:

- Conjoint Analysis with Logit Model
- RFM Analysis
- Cluster Analysis
- CLV Calculation
- Market Basket Analysis
- Linear Regression
- Nonlinear Regression (Log-log Model)
- Random Walk
- Market Model
